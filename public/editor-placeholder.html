⚠️ Редактор не подключен

Редактор карт (PolyGenStudio) — это отдельное приложение.
В данный момент оно не настроено для онлайн-версии.

Как исправить:
  1. Задеплойте папку PolyGenStudio-main как отдельный проект на Vercel.
  2. Скопируйте полученный URL (например https://my-editor.vercel.app).
  3. Добавьте переменную в настройки основного проекта Protocol TX:
VITE_EDITOR_URL=https://my-editor.vercel.app

Для разработчиков: локально используйте npm run start:all чтобы запустить и игру, и редактор.
