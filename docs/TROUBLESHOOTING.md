# Руководство по устранению неполадок

## Проблемы с запуском

### Черный экран при запуске игры
**Решение**: Эта проблема была исправлена в последней версии. Если проблема сохраняется:
1. Проверьте консоль браузера на наличие ошибок
2. Убедитесь, что используется последняя версия кода
3. Очистите кэш браузера (Ctrl+Shift+Delete)

### Ошибка "No camera defined"
**Решение**: Исправлено. Камера теперь создается автоматически при необходимости.

### Ошибка "Cannot read properties of undefined"
**Решение**: Исправлено. Все подписки на события физики теперь происходят после инициализации сцены.

## Проблемы с физикой

### Танк трясется при движении
**Решение**: Исправлено в последней версии. Проблема была связана с рассинхронизацией между физическим телом и визуальным мешем. Теперь:
- Камера обновляется после обновления физики
- Используется `getAbsolutePosition()` для правильной синхронизации
- Улучшена синхронизация позиции меша с физическим телом

### Танк "проваливается" сквозь землю
**Решение**: 
1. Проверьте настройки физики в Physics Panel (F3)
2. Убедитесь, что `hoverHeight` установлен правильно
3. Проверьте, что `hoverStiffness` и `hoverDamping` имеют разумные значения

### Танк слишком нестабилен
**Решение**:
1. Откройте Physics Panel (F3)
2. Увеличьте `uprightForce` и `uprightDamp`
3. Увеличьте `movementDamping`
4. Уменьшите `hoverStiffness` если танк "прыгает"

## Проблемы с производительностью

### Низкий FPS
**Решение**:
1. Уменьшите `renderDistance` в настройках
2. Закройте Debug Dashboard (F2) если он открыт
3. Проверьте, что используется WebGPU (не WebGL2)
4. Уменьшите количество врагов на карте

### Лаги при движении
**Решение**:
1. Проверьте консоль на наличие ошибок
2. Убедитесь, что используется последняя версия
3. Попробуйте уменьшить настройки графики

## Проблемы с управлением

### Камера не следует за танком
**Решение**: 
1. Нажмите C для центрирования камеры
2. Проверьте, что камера не заблокирована
3. Перезапустите игру если проблема сохраняется

### Башня не поворачивается
**Решение**:
1. Проверьте, что используется мышь (не клавиатура)
2. Нажмите Z/X для клавиатурного управления башней
3. Проверьте настройки чувствительности мыши

## Проблемы с сетью (если применимо)

### Не могу подключиться к серверу
**Решение**: Сетевая игра находится в разработке. На данный момент доступен только одиночный режим.

## Получение помощи

Если проблема не решена:
1. Проверьте [CHANGELOG.md](../CHANGELOG.md) на наличие известных проблем
2. Создайте issue в репозитории GitHub с описанием проблемы
3. Приложите скриншоты и логи из консоли браузера (F12)
